# aml-env-setup
Few scripts to setup a comfortable cmd environment on Azure ML compute instance

### Installed tools
- Fish shell
- ranger
- tmux
- neovim
    - aliased to `vi`
- Fuzzy finder fzf
- fd

## How to setup AML instance
### SSH config
Configure `azure-ml` SSH alias in your `~/.ssh/config`

### Running the setup
`$ ./setup_aml_env.fish`
